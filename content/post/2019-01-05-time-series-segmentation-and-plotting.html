This post is going to analyse a component of the data downloaded here, plotting the slope of the yield curve and the S&P 500 index. We will also discretise the yield curve into bins or categories based on its level and change, plotting a histogram of subsequent S&P 500 returns associated with each bin. The aim of this excercise is to determine if the slope of yield curve leads the stock market.

Background

The yield curve, calculated as the spread between the federal funds rate and 10 year treasury note rate has been theorised to be a powerful predictor of the business cycle (wikipedia reference). We are not interested in the business cycle, but rather if the yield curve is a leading indicator to the stock market. Specifically, does any particular level, change (or combination thereof) of the yield curve provide information about future stock market returns. For example if the yield curve is in the lowest third of its historical range, and is increasing, do subsequent stock market returns differ compared to other periods?

The logic for using a discretisation approach, as opposed to interpreting regression co-efficients is motivated by the expectation of non-linearities in the relationship between the yield curve and stock market. If the inversion of the yield curve is considered a signal of lower future economic activity as theorised above, then it would be expected that the status of the yield curve at other times is not relevent.

Data preparation

We begin loading the required packages and the data prepared previously.

library("tidyverse")
library("cowplot")
library("caret")
library("scales")

econ_fin_data <- readRDS("econ_fin_data.Rda")

Next, we calculate the yield curve, subtracting the fed funds rate from the 10 year treasury note rate. We then categorise the yield curve into bins representing specific level and change values. The code block below decomposes the time series of the yield curve into 6 bins. The level of the yield curve is split into terciles or three bins containing an equal number of months. The change in level is categorised as either an increase or decrease (positive or negative 6 month change). The combination of these is then lagged 6 months and 12 months resulting in 18 bins in total.

# create the yield curve time series (designated "ff_10")
x2 <- econ_fin_data %>% mutate(ff_10 = GS10 - FEDFUNDS) %>%
  
  # select data required, including indicator under analysis
  select(date, close, fwd_rtn_m, ff_10) %>% 
  
  # lagged values of indicator under analysis
  mutate(x1.lag6  = lag(ff_10, 6), 
         x1.lag12 = lag(ff_10, 12),
         
         # tercile level factor
         x1.qntlx = ntile(ff_10, 3), 
         x1.qntl = case_when(x1.qntlx == 1 ~ "_low", 
                             x1.qntlx == 2 ~ "_mid", 
                             x1.qntlx == 3 ~ "_high"),
         
         # change in level indicator
         x1.rtn6  = ff_10 - x1.lag6,
         x1.rtn12 = ff_10 - x1.lag12,
         
         # binary change in level factor
         x1.delta = if_else(ff_10 > lag(ff_10, n = 6), 
                            "incr", 
                            "decr")) %>% 
  
  # factor combining tercile level and binary change in level factors 
  unite(x1_lag00, c(x1.qntl, x1.delta),sep = "_", remove = FALSE) %>%
  
  # lagged combined factor and filter out NA's
  mutate(x1_lag06 = lag(x1_lag00, 6),                          
         x1_lag12 = lag(x1_lag00, 12)) %>%                           
  filter(!is.na(x1.lag12))

Three strings are returned, the current and two lagged values. Note the format of these strings, this will be used later in the the histogram. The first five characters specify whether the indicator is current or lagged, the middle three characters specify the level tercile, and lastly we have the change in slope indicator.

Recall that we would like to plot a histogram of subsequent S&P500 returns. In order to enable segmentation for the histogram we need to create dummy variables for each of the 18 bins. This is were the caret package comes in handy.

# dummy variables for each (current & lagged) combined level / change factor
x3 <- predict(dummyVars(" ~ x1_lag00", data = x2), newdata = x2)
x4 <- predict(dummyVars(" ~ x1_lag06", data = x2), newdata = x2)
x5 <- predict(dummyVars(" ~ x1_lag12", data = x2), newdata = x2)

# combine dummy variable sets (current and lagged) to single data frame 
x6 <- as.tibble(cbind(x3, x4, x5)) %>% select(-contains("NA")) %>% 
  rownames_to_column(var = 'rowIndex') %>% 
  
  # transform combined dummy variable data from wide to long format
  gather(key = 'Indicator', value = 'Value', -rowIndex) %>% 
  
  # convert dummy variable to factor
  mutate(Value_fact = ifelse(Value == 1, "In", "Out"))

# assign rownames to columns in order to join return data to dummy variable data 
x7 <- x2 %>% select(date, fwd_rtn_m) %>% rownames_to_column(var = 'rowIndex')

# data for histogram plot - join return data to dummy variable data 
x8 <- full_join(x6, x7, by  = 'rowIndex') %>% 
  # rename indicator
  mutate(Indicator = str_replace(Indicator, "x1_", "ff_10 : "))

We now have a table containing 18 records for each date, six for each of the current and two lagged level / change bins as previously defined. These are our dummy variables. Three of these will be labelled “In” (one of the six of each of the current and two lagged values), representing the status of the yield curve at that point in time. The table also contains the monthly return of the S&P 500 for the month subsequent to that labelled. It is from this table that the histogram is derived.

Measure of dissimilarity

We are interested in the predictive power of yield curve to the stock market. Should there be a relationship between the status of the yield curve defined by our level / change bins and future returns, returns subsequent to when specific bins are triggered will differ to all other returns.

How do we determine if stock market returns do indeed differ for specific periods, in our case, periods when the yield curve is in a specific decomposition bin with reference to its level and change, and all other times? We could compare the mean monthly return for each period, testing for significance of difference in mean with a T test. The T test however assumes the underlying data is normally distributed. This is not the case with returns of financial assets. We therefore need to use a non parametric test, a test which does not require the normality assumption. The Kolmogorov-Smirnov test is one such test. From Wikipedia

“The two-sample K-S test is one of the most useful and general nonparametric methods for comparing two samples, as it is sensitive to differences in both location and shape of the empirical cumulative distribution functions of the two samples.”

The code below creates a nested data frame so that a K-S test can be performed for each bin, assessing the dissimilarity of subsequent monthly returns when that bin is triggered and when not. The difference in average return is also calculated. These measurements will be used to narrate the histogrm plot.

# data for kolmorogov smirnov test - list of data frames for
# each value of each (current & lagged) combined level / change factor
x8.1<-x8 %>% select(Indicator, date, Value_fact, fwd_rtn_m) %>% 
  spread(Value_fact, fwd_rtn_m) %>% nest(-Indicator)

# perform ks test, map to each element of nested dataframe
x8.2<-x8.1 %>% mutate(ks_fit = map(data, ~ks.test(.$In, .$Out)),
                      p_val  = map_dbl(ks_fit, "p.value"))

# mean return data & difference in mean for histogram text
x9 <- x8 %>% group_by(Value_fact, Indicator) %>% summarise(Mean = mean(fwd_rtn_m))
x9.1<-x9 %>% spread(Value_fact, Mean) %>% mutate(mean_diff = In - Out)

So what are these plots telling us about the yield curve? Does the yield curve predict stock market returns?

Based on the histogram results, when the current value of the yield curve (defined as the fed funds rate less the 10 year treasury note rate) is low and increasing (the facet labelled “lag00_low_incr”) we want to stay out of the market, the average of subsequent months returns are nearly 1% lower than all other periods in this situation. We are confident this is a significant difference based on the results of the two sample Kolmorogov Smirnov test, this test returns a p value of 0.6% (the null hypothesis that the two samples were drawn from the same distribution is rejected if the p-value is less than your significance level - typically set at 5%).

If we look at the plot of the yield curve we can confirm this visually. It does appear for example that the drawdowns in years 2000 and 2008 are preceded preceded by an increase in the yield curve from a low level. The second most significant bin for the current value of the yield curve is when it is in the mid tercile and decreasing, subsequent returns are 0.75% higher with a K-S p value of 3.7%. For reference, average monthly returns over the time under analysis are 0.61%.

Looking at lagged values of the yield curve, four situations arise that would lead us to take a view on the market:
1. When the 6 month lagged yield curve is low and increasing (lower subsequent returns)
2. When the 12 month lagged yield curve is high and increasing (higher subsequent returns)
3. When the 12 month lagged yield curve is low and decreasing (lower subsequent returns)
4. When the 12 month lagged yield curve is mid and decreasing (higher subsequent returns)

Limitations

Any analysis technique has it’s advantages and disadvantages. Discretisation or binning is useful in that it identifies non-lineararities, removes outliers and is simple to interprete. The approach as it is used above does have some disadvantages however. It is quiet crude in that by design it peaks into the future. The measurement of the tercile barriers during the 1970’s uses data from the 2000’s for example. A more robust approach would be to utilise a rolling fixed starting point window to estimate terciles for binning. Further, it may be that tercile bins are not appropriate cutoffs, maybe the traditional inversion of the yield curve or some other level is an appropriate cut-off. We have not investigated this as a part of the current analysis. Despite these limitations, this approach does provide a high level overview of the relationship between the level of the yield curve and subsequent stock market returns. This information may be useful in building more sophisticated models, informing relevant dummy variables in a regression for example.
